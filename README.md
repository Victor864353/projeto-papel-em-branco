# Projeto Papel em Branco 📝

O **Projeto Papel em Branco** é uma aplicação web criada para ajudar artistas e também mestres e jogadores de RPG a criar rapidamente novas ideias, seja para arte ou para campanhas.

## 🌐 Acesso Online

A aplicação está disponível em produção via GitHub Pages:

https://Victor864353.github.io/projeto-papel-em-branco/

## ✨ O que a aplicação faz
A ferramenta web é dividida em duas sessões:

**Gerador de Ideias:**
Permite, através da aleatoriedade, criar ideias completamente novas que fogem do convencional.

Categorias:
- **Personagem**
- **Cenários**
- **Criaturas**
- **Props**
- **Estudos** (um extra ainda em desenvolvimento)

**Criação**
Oferece total liberdade para o usuário escolher e escrever suas ideias.

Dividido em quatro áreas de criação:
- **Personagem** 
- **Criatura** 
- **Cenário** 
- **Props** 

## 💾 Como salvar o conteúdo

Cada área permite exportar o material criado usando:

- **TXT:** salva em texto simples para copiar ou usar em outros editores.
- **JPEG:** gera imagem da ficha usando `html2canvas`.
- **PDF:** cria um documento formatado usando `jsPDF`.

## 📁 Estrutura do projeto

Arquivos principais:

- `index.html` – página inicial.
- `criacao/` – interface de criação com páginas para personagem, criatura, cenário e props.
- `dados/` – dados para os sorteadores da sessão de geração de ideias.
- `ideias/` – páginas de inspiração com exemplos de cenários, criaturas, estudos, personagens e props.
- `criacao/salvamento/` – scripts para exportação de conteúdo.

## 🛠 Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript puro
- `html2canvas`
- `jsPDF`

## 🚀 Como usar localmente

1. Baixe ou clone o repositório.
2. Abra `index.html` no navegador.
3. Navegue até as páginas de criação para gerar e exportar suas ideias.
